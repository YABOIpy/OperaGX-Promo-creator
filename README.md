**Config**

| Name | Type | Description | 
| ---  | ---  | ---         |
| `threads` | CONFIG | Amount Of Instances u want to run
| `thread_limit` | CONFIG | the limit of concurrent threads allowed to run 0 = no limit
| `proxy` | HTTP | Proxy Address: user:pass@ip:port
